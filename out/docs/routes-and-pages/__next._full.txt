11:T741,<h1 id="routes-and-pages">Routes and Pages</h1><p>This document covers the current route surface.</p>
<h2 id="public-pages">Public pages</h2><ul>
<li><code>/site</code>: open source website route</li>
<li><code>/docs</code>: docs index</li>
<li><code>/docs/[slug]</code>: public markdown docs</li>
<li><code>/login</code>: team login page</li>
</ul>
<h2 id="protected-pages">Protected pages</h2><ul>
<li><code>/</code>: authenticated workspace entry</li>
<li><code>/integrations</code>: incoming webhook management</li>
<li><code>/channel/[id]</code>: channel view</li>
<li><code>/dm/[id]</code>: direct message view</li>
<li><code>/tasks</code>: task board</li>
<li><code>/settings</code>: user settings</li>
</ul>
<p>Additional protected product routes are expected later as more of the app shell is implemented.</p>
<h2 id="api-routes">API routes</h2><ul>
<li><code>/api/auth/[...nextauth]</code><ul>
<li>NextAuth route handler for login and callbacks</li>
</ul>
</li>
<li><code>/api/health</code><ul>
<li>checks Postgres and MinIO connectivity</li>
</ul>
</li>
<li><code>/api/integrations/webhooks</code><ul>
<li><code>GET</code>: list current webhooks and channels for an authorized operator</li>
<li><code>POST</code>: create a new incoming webhook</li>
</ul>
</li>
<li><code>/api/integrations/webhooks/[id]</code><ul>
<li><code>DELETE</code>: revoke a webhook</li>
</ul>
</li>
<li><code>/api/hooks/[token]</code><ul>
<li><code>POST</code>: accept message delivery from an external application</li>
</ul>
</li>
</ul>
<h2 id="route-protection">Route protection</h2><p><code>proxy.ts</code> keeps the public surface small. App pages outside the known public routes require an authenticated user session.</p>
