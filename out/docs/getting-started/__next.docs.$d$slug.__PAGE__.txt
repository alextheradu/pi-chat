3:T9e9,<h1 id="getting-started">Getting Started</h1><p>Pi-Chat is an open-source, self-hosted Slack alternative. This page covers what you need to run it locally.</p>
<h2 id="prerequisites">Prerequisites</h2><ul>
<li>Node.js 20 or later</li>
<li>Docker and Docker Compose (for local Postgres and MinIO)</li>
<li>A Google Cloud project with OAuth credentials</li>
</ul>
<h2 id="clone-the-repository">Clone the repository</h2><pre><code class="language-bash">git clone https://github.com/alextheradu/pi-chat.git
cd pi-chat
</code></pre>
<h2 id="install-dependencies">Install dependencies</h2><pre><code class="language-bash">npm install
</code></pre>
<h2 id="configure-environment-variables">Configure environment variables</h2><pre><code class="language-bash">cp .env.example .env
</code></pre>
<p>Open <code>.env</code> and fill in the required values:</p>
<pre><code class="language-env"># Database
DATABASE_URL=&quot;postgresql://user:password@localhost:5432/pichat&quot;

# NextAuth
NEXTAUTH_URL=&quot;http://localhost:3001&quot;
NEXTAUTH_SECRET=&quot;your-secret-here&quot;

# Google OAuth
GOOGLE_CLIENT_ID=&quot;your-client-id&quot;
GOOGLE_CLIENT_SECRET=&quot;your-client-secret&quot;

# Access policy
ALLOWED_DOMAIN=&quot;yourteam.org&quot;
ADMIN_EMAIL=&quot;admin@yourteam.org&quot;

# MinIO / S3 storage
MINIO_ENDPOINT=&quot;localhost&quot;
MINIO_PORT=&quot;9000&quot;
MINIO_ACCESS_KEY=&quot;minioadmin&quot;
MINIO_SECRET_KEY=&quot;minioadmin&quot;
MINIO_BUCKET=&quot;pichat&quot;
</code></pre>
<h2 id="start-local-services">Start local services</h2><p>Spin up PostgreSQL and MinIO with Docker Compose:</p>
<pre><code class="language-bash">docker compose -f docker-compose.dev.yml up -d
</code></pre>
<h2 id="run-database-migrations">Run database migrations</h2><pre><code class="language-bash">npm run db:migrate:deploy
</code></pre>
<h2 id="seed-initial-data">Seed initial data</h2><pre><code class="language-bash">npm run db:seed
</code></pre>
<h2 id="start-the-development-server">Start the development server</h2><pre><code class="language-bash">npm run dev
</code></pre>
<p>The app starts on <code>http://localhost:3001</code> by default. Open it in a browser and sign in with a Google account that matches <code>ALLOWED_DOMAIN</code>.</p>
<h2 id="verify-dependencies-are-healthy">Verify dependencies are healthy</h2><pre><code class="language-bash">npm run test:db
npm run test:minio
</code></pre>
<p>Both should report success. If not, check that Docker containers are running and your <code>.env</code> values are correct.</p>
